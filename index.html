Experimental port of SICMUtils into a self-hosted ClojureScript widget.

The source is the Javascript in this page and compliled ClojureScript for repl.core. There is also a repl.macros library and a repl.eval namespace used for evaluation. The ClojureScript code calls cljs.js/eval-str to evaluate the contents of the textarea input field in the repl.eval namespace. Most of sicmutils.env is implicitly included in the repl.eval namespace, but some of the macros are still missing. It then uses repl.eval/pTeX to send the result back to the JS side which formats the string with MathJax.

Code is available at github.com/sigmaxipi/sicm-repl.
